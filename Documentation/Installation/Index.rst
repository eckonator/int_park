============
Installation
============

Target group: **Administrators**

1. Install this extension via composer ``composer require onm/int-park`` if you have non-composer installation then just download from TER and install it.
2. Include typoscript in template
3. Under any folder, create a db record 'Park', add image in it.
4. Using a backend module, you can place markers on the image.

.. Note::

   jQuery is needed to run this.

.. important::

   Font css path has to be added in extension configurations to use icons inside markers. (Pro feature)
